# JSMarcos 产业自动化机器人

JSMarcos 是一个可用JS代码控制游戏运行的脚本mod，并可与其他mod联动，除了开发文档写的很简略外就没有其他缺点了。

# 安装方法

1. 安装mod加载器fabric或forge，这具体过程略了

2. 下载mod并安装，注意选择正确的版本和加载器 https://github.com/JsMacros/JsMacros/releases/

3. 把脚本里的配置改成你自己的
